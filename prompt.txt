git clone https://github.com    "la repository"
cd galleria-artista

Crea l'ambiente: python -m venv venv
Attivalo: .\venv\Scripts\activate
Installa le librerie: pip install -r requirements.txt

Crea il Database:
python manage.py makemigrations
python manage.py migrate

Crea il tuo utente: python manage.py createsuperuser (inserisci nome e password).
Avvia Django: python manage.py runserver

Apri un secondo terminale (lascia il primo acceso con Django) e vai nella cartella 
frontend:
Entra nella cartella: cd frontend
Scarica i pacchetti: npm install 
(questo installerà Three.js, React-Three-Fiber e tutto il necessario 
leggendo dal file package.json).
Avvia React: npm start
